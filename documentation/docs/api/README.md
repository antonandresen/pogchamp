## Installation

Using npm:

```bash
  npm install pogchamp
```

Using yarn:

```bash
  yarn add pogchamp
```

Using pnpm:

```bash
  pnpm add pogchamp
```

## Features

- ESM & commonJS import support
- Well optimized helpful utility functions

## Usage/Examples

```javascript
import { hasValue, invertObject } from 'pogchamp'

hasValue(true)
invertObject({ name: "anton"})
```
